# Finance Monitoring AI System

Build the foundation of the Finance Monitor AI system by completing all planning, documentation, system design, and technical preparation needed before development begins.

## Overview
This repository contains planning, design, and initial tooling for the Finance Monitoring AI System. The project will include components for data ingestion, anomaly detection, alerting, and a dashboard for financial monitoring.

## Quick start
1. Clone the repo:

   git clone https://github.com/Migspajela23/Finance-Monitoring-AI-System.git
   cd Finance-Monitoring-AI-System

2. (Optional) Create a virtual environment and install dependencies:

   python -m venv .venv
   source .venv/bin/activate  # macOS/Linux
   .\.venv\Scripts\activate   # Windows
   pip install -r requirements.txt  # if present

3. Run tests (when available):

   pytest
